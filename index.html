What I Do

Embedded Hardware Development

I develop embedded hardware using all the Stages of embedded system development process from Ideation to Product Release. This stages help me to develop the hardware in a way where i can make the product much more efficient & upgradeable.

Embedded Software Development

I strive to develop and implement scalable software using Embedded C & Python which gives me the privledge to get maximum performance from the hardware.

PCB Design

Using my 4+ years of experience in embedded domain, i design the PCB & Schatmetics usinf EASYEDA software. I can develop multi layer pcb which is space optmised & high quality.

Embedded Testing

Beyond coding, I also do the testing of the hardware & the software to make sure everything is performing as per the design. I use VectorCast tool & GDB debugger to perform the testing.
